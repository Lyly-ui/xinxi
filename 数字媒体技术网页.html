颜色理论与应用

色彩模型

数字媒体中常用的色彩模型包括RGB（加色模型）、CMYK（减色模型）和HSV/HSL（色调、饱和度、亮度）。了解不同模型的应用场景是数字媒体设计的基础。

初级

对比度与可读性

合理的对比度设计可以提升内容的可读性和视觉层次。WCAG 2.1标准建议文本与背景的对比度至少达到4.5:1，大文本至少3:1。

中级

配色方案

常见的配色方案包括单色、类比色、互补色、分裂互补色和三色组合。选择恰当的配色方案可以传达不同的情感和信息。

初级

图形与排版

矢量与位图

矢量图形由数学公式定义，可无限放大不失真；位图由像素组成，适合照片等复杂图像。了解两者的区别和适用场景是数字媒体设计的关键。

初级

字体与排版

字体选择、行高、字间距、段落间距等排版要素直接影响内容的可读性和视觉美感。掌握排版原则是数字媒体设计的基本功。

中级

网格系统

网格系统为设计提供结构和一致性，帮助创建平衡、和谐的布局。在网页设计和UI设计中，网格系统尤为重要。

使用CSS创建简单动画的示例：

/* 定义关键帧动画 */
@keyframes fadeInScale {
  0% {
    opacity: 0;
    transform: scale(0.8);
  }
  100% {
    opacity: 1;
    transform: scale(1);
  }
}

/* 应用动画到元素 */
.animated-element {
  animation: fadeInScale 0.5s ease-out forwards;
  animation-delay: 0.2s;
}

这个示例展示了CSS关键帧动画的基本用法，这是数字媒体中创建流畅界面动画的常用技术。
